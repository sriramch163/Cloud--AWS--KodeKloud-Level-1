Steps to Complete the Task Using the AWS Console

Follow these steps carefully.

Step 1: Log in to AWS Console

Open the provided Console URL.

Enter your assigned username and password.

Verify the region in the top right corner is:
N. Virginia (us-east-1).

Step 2: Navigate to the VPC Service

In the AWS search bar, type VPC.

Select the VPC service from the results.

Step 3: Start Creating a VPC

In the left navigation menu, click Your VPCs.

Click the Create VPC button.

Step 4: Configure the VPC

On the Create VPC page, input:

Basic Settings

Resource type:
Select VPC only.

Name tag:
Enter
xfusion-vpc

IPv4 CIDR Block

Leave as default (10.0.0.0/16) unless the task specifies otherwise.

Enable IPv6

This is the key requirement for this task.

Under IPv6 CIDR block, select:
Amazon-provided IPv6 CIDR block

AWS will automatically assign a /56 IPv6 CIDR block to the VPC.

All other fields can remain default unless instructions require changes.

Step 5: Create the VPC

Scroll to the bottom.

Click Create VPC.

Wait for the success notification:
“VPC successfully created.”

Step 6: Validate the VPC

Return to Your VPCs and verify:

VPC name = xfusion-vpc

Region = us-east-1

IPv6 CIDR Block = Amazon-provided (visible under the VPC details)

VPC state = Available

This confirms successful configuration.
