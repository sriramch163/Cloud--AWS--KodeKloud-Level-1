The objective of this task is to create a Virtual Private Cloud (VPC) named `datacenter-vpc` in the `us-east-1` (N. Virginia) region with any valid IPv4 CIDR block.

This task supports the Nautilus DevOps team's phased AWS migration by establishing isolated, configurable network infrastructure.

The created VPC must:

- Exist only in the `us-east-1` region.
- Be named `datacenter-vpc`.
- Include any valid IPv4 CIDR block (example: 10.0.0.0/16 or 192.168.0.0/16).
